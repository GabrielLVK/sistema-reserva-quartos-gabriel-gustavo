# Sistema de Reserva de Quartos - EcoStay

## Tema Escolhido
Sistema de reserva de quartos de hotel/pousada.

## Objetivo do Sistema
Oferecer uma plataforma web simplificada onde hóspedes possam visualizar quartos disponíveis e realizar reservas, e administradores possam gerenciar os quartos e acompanhar as reservas efetuadas.

## Descrição Geral do Funcionamento
O sistema possui duas visões. A visão pública permite que qualquer visitante veja os quartos disponíveis. Para realizar uma reserva, o usuário deve se cadastrar e fazer login (área privada). O usuário logado pode escolher um quarto, definir as datas de entrada/saída e confirmar a reserva. O sistema valida se o quarto não está ocupado no período selecionado.

## Público-Alvo
- **Hóspedes:** Pessoas que buscam acomodação e desejam reservar quartos online de forma rápida.
- **Administradores:** Gerentes do estabelecimento que controlam a ocupação e o cadastro dos quartos.

## Funcionalidades Principais
1. Cadastro e Autenticação de Usuários (com controle de sessão).
2. Listagem de quartos disponíveis com filtros básicos.
3. Criação, visualização e cancelamento de reservas por parte do usuário.
4. Gerenciamento de quartos (Criar, Ler, Atualizar, Deletar - CRUD) exclusivo para administradores.
